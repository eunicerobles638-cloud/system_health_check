# System Health Check Utility
A lightweight script for Termux to monitor disk space and verify file permissions efficiently.

## Features
* **Disk Monitoring**: Checks available storage space.
* **Permission Check**: Verifies read/write/execute permissions for critical files.
* **Automated Sync**: Includes a shortcut for fast GitHub updates.

## How to Use
Run the main script using:
```bash
./perm_check.sh
